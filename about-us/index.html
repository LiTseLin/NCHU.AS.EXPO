About wpkoi themes

Different Layouts

You can choose from different body, header, footer and blog styles with WPKoi theme.

Plugin Compatible

WPKoi themes are compatible with Elementor page builder and WooCommerce webshop plugins!

GPL License

The theme and the plugin is under GPL license so You can use them on as many sites as You want!

Premium Features

Unlimited Colors

You can customize almost all of the colors in the customizer. Header, footer, content, sidebars, font colors etc.

More Layouts

However the free theme has a lot of layout feature but the premium version has much more!

Hooks

You can add custom HTML, JS or PHP code to wp_2_head, wp_2_footer or to oder parts of the theme.

Customizable Fonts

Customize the typography with Premium version. You can select from Google Fonts and add the paramaters.

Page Header

Add images or video to the header of the pages and blog loops with this theme! You can also add texts on it.

Much More...

If You need a fully customizable stable and fast theme with clean design WPKoi themes are for You!
